OpenAI O3 pro的一些反馈

OpenAI 最近推出了 GPT-3 Pro，这是他们迄今为止最强大的 AI 模型。

GPT-3 Pro 是 GPT-3 的一个升级版本。GPT-3 是一个推理模型，在今年早些时候发布。与传统的 AI 模型不同，推理模型能够一步一步地解决问题，这使得它在物理、数学和编程等领域表现更加可靠。 从本周二开始，GPT-3 Pro 将面向 ChatGPT Pro 和团队用户开放，取代之前的 GPT-1 Pro 模型。企业用户和教育用户将在下周获得访问权限。同时，GPT-3 Pro 也已经可以在 OpenAI 的开发者 API 中使用了。

在 API 使用方面，GPT-3 Pro 的定价是：每百万个输入标记收费 20 美元，每百万个输出标记收费 80 美元。输入标记是指输入到模型中的内容，输出标记则是模型根据输入生成的内容。 o3 系列价格下调 80%，当前定价为每百万输入/输出令牌 2/8 美元（此前为 8/40 美元），并为缓存输入令牌提供 75% 的折扣。o3 的每令牌价格与 GPT-4.1 相当，与 Gemini 2.5 Pro 和 Claude 4 Sonnet 价格相近，但比 Claude 4 Opus 便宜 8 倍。

一百万个输入标记大约相当于 75 万个单词，比《战争与和平》这本书还要长一些。

o3-pro 使用与 o3 相同的架构，但通过生成更长的隐性思考链（hidden reasoning chain）显著提升了推理效能，O3 Pro 在所有测试类别中都优于 O3，特别是在科学、教育、编程、商业和写作帮助等关键领域。评审人员还认为 O3 Pro 在清晰度、全面性、指令遵循和准确性方面都表现更好。

O3 Pro 具备多种工具使用能力，包括：

  • 网络搜索
  • 文件分析
  • 视觉输入处理
  • Python 编程
  • 个性化响应（利用记忆功能）

o3-pro 在处理长上下文和复杂对话时表现更强，能够更深入地理解和整合输入信息。Ben Hylak 在试用报告中用“上帝渴求上下文”类比，强调其对上下文的强大依赖和处理能力：

  • 长文档分析：在处理长篇技术文档、法律合同或研究论文时，o3-pro 能更准确地提取关键信息并生成摘要。
  • 多轮对话：在需要维持长时间对话连贯性的场景（如客户支持、咨询服务），o3-pro 能更好地追踪上下文，避免信息丢失。
  • 创意写作：在小说或剧本创作中，o3-pro 能根据复杂的背景设定生成更连贯、逻辑一致的故事线。

o3-pro 在输出内容的清晰度（clarity）和全面性（comprehensiveness）上优于 o3，特别是在需要高质量输出的场景中。在实时问答或在线教育平台中，o3-pro 的快速响应和高质量输出提升用户体验
